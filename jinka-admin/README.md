# Jinka City Administration - Admin Panel

Modern, beautiful, and responsive admin panel for Jinka City Administration built with React, Ant Design, and Refine.

## 🎨 Features

- ✅ Modern and beautiful UI design
- ✅ Fully responsive (mobile, tablet, desktop)
- ✅ Dashboard with statistics and recent activities
- ✅ Citizens management
- ✅ Departments management
- ✅ Announcements system
- ✅ Projects tracking
- ✅ Events calendar
- ✅ Documents management
- ✅ Reports and analytics
- ✅ Messaging system
- ✅ Settings page
- ✅ Multi-language support (English/Amharic)
- ✅ Dark sidebar with gradient branding
- ✅ Search functionality
- ✅ Notifications
- ✅ User profile management

## 🛠️ Technology Stack

- **Frontend Framework**: React 18
- **Build Tool**: Vite
- **UI Library**: Ant Design 5
- **Admin Framework**: Refine
- **Routing**: React Router v6
- **HTTP Client**: Axios
- **Date Handling**: Day.js
- **Icons**: Ant Design Icons

## 📋 Prerequisites

- Node.js 16+ and npm/yarn
- API endpoint from the backend team

## 📱 Pages

1. **Dashboard** - Overview with statistics and recent activities
2. **Citizens** - Manage citizen records
3. **Departments** - Manage city departments
4. **Announcements** - Create and manage announcements
5. **Projects** - Track city projects
6. **Events** - Calendar view of events
7. **Documents** - Document management system
8. **Reports** - Analytics and reports
9. **Messages** - Internal messaging
10. **Settings** - System and user settings
